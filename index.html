microcosm

Decentralizing atproto with self-hosted services that are cheap to run and work at bluesky scale.

Project status: extremely work-in-progress

Self-host ATProtocol building blocks

Constellation

A global atproto backlink index

  • Self host: handles the full atproto firehose write throughput on a raspberry pi.
  • Storage efficient: less than 2GB/day disk consumption.
  • Privacy-conscious: handles record deletion and account de/re-activation/deletion.
